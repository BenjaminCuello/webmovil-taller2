# Frontend - InfoMóvil Taller 2

Este directorio contiene la aplicación web móvil (HTML + Tailwind + JS puro) que se empaqueta con Apache Cordova.

El directorio incluye el HTML principal, los estilos generados con Tailwind y los archivos JavaScript que manejan las vistas, las búsquedas y la comunicación con las APIs del backend.

## Configuración de APIs

El frontend está pensado para consumir **solo** las APIs propias del Taller 2:

- NestJS → `/pokemon` (lista + detalle)
- Express → `/countries` y `/countries/search`
- FastAPI → `/weather?city=` y `/holidays/{countryCode}/{year}`

## Docker Compose

```bash
# Desde la raíz del repositorio
docker compose up --build
# Frontend disponible en http://localhost:8080
```

Si es necesario ajustar las URL de las APIs (por ejemplo, al usar un emulador Android o una IP local), esto se realiza en `frontend/www/js/config.js`.
