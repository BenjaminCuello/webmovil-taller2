# Cordova / APK Android

Esta carpeta contiene el proyecto Cordova utilizado para generar el APK de la aplicación InfoMóvil.
Aquí se empaqueta el frontend desarrollado en el Taller 2 y se prepara para ejecutarse como aplicación Android.

---

## Contenido

- `www/`: copia del frontend listo para empaquetar
- `config.xml`: configuración general de la app
- `platforms/android/`: archivos generados para compilar el APK
- Scripts y configuraciones necesarias para que la app pueda consumir las tres APIs del backend

---

## APK generado

El APK final (modo debug) se encuentra en:

cordova/platforms/android/app/build/outputs/apk/debug/app-debug.apk
